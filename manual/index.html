01 // CONCEITO E VISÃO

VERTIC: O Ecossistema Integrado

A VERTIC é uma holding integradora no modelo Full Stack Advisor — o Sistema Operacional do empresário de alto valor (HNWIs). A holding é o portal/núcleo; as entregas acontecem em quatro vértices autônomas, cada uma com cor e território próprios, herdando o DNA visual da marca-mãe.

  • Vertic Marketing — captação, funis, branding e tráfego (a demanda).
  • Vertic Consultoria — diagnóstico e mapeamento de processos (descobre o problema).
  • Vertic Tech — plataformas sob medida e automações (constrói a solução).
  • Vertic Wealth — planejamento patrimonial e investimentos (protege e multiplica).

O ciano é reservado exclusivamente para a holding VERTIC; nenhuma vértice o utiliza como assinatura. A PGR LAW é o braço jurídico parceiro da Wealth — empresa separada, não uma subsidiária.

02 // ESTRUTURA VISUAL

A Estética Finance Noir & Depth UI

A identidade afasta-se do visual corporativo comum e aproxima-se de softwares técnicos de alta performance: planos de fundo pretos-azulados profundos, layouts divididos por bordas finas de micro-opacidade, micro-interações simétricas e vazamentos sutis de luz radial (radial glow).

Sobre essa base escura, cada vértice projeta sua cor-assinatura. Duas vértices (Marketing e Wealth) operam em tema claro e usam a transição The Shift (seção 07) para dialogar com o Noir da holding.

05 // GEOMETRIA OFICIAL

Os Dois Símbolos V-Node

Existem dois marks distintos, com papéis diferentes. Geometria validada a partir do código real de produção (viewBox 0 0 100 100). A descrição antiga ("inclinação 2, nó em 60,80, raio 9") estava incorreta e foi descartada.

Mark A — V-Node Icon (logotipo / lockup)

V único de traço grosso. Usado em logos, headers, favicons e nos lockups de todas as vértices. Haste sempre Slate Gray; nó na cor da vértice.

<path d="M20 25 L50 75 L80 25" stroke-width="8" stroke-linecap="round"/> <circle cx="50" cy="75" r="10"/> /* haste: #71717a · nó: cor da vértice */

Mark B — Triple V-Node (sistema / decorativo)

Três Vs aninhados (não-paralelos: inclinações 1.62 / 1.83 / 2.25, afunilando para dentro). Usado em preloader, loading screen e watermark do hero. O nó fica no vértice do V médio.

<path d="M10 20 L50 85 L90 20" stroke-width="2"/> // externo <path d="M20 20 L50 75 L80 20" stroke-width="1.5"/> // médio (acento) <path d="M30 20 L50 65 L70 20" stroke-width="1"/> // interno <circle cx="50" cy="75" r="4"/> // nó no vértice do médio
  • Preloader (escuro): externo branco, médio ciano, interno branco, nó ciano. Anima por traços desenhando + nó crescendo.
  • Watermark hero (escuro): externo branco tracejado em baixa opacidade, médio e interno brancos, nó ciano.
  • Versão clara: hastes navy #1e293b, médio e nó em Deep Cyan #0e7490.

07 // TRANSIÇÕES DE TEMA

The Shift

Conceito-guarda-chuva com dois membros direcionalmente opostos. Wealth ascende para a luz; Marketing desce para a sombra — mesma engine, vetores contrários.

The Ascension

Wealth. Escuro/ciano → claro/dourado. O cliente sobe ao status de patrimônio protegido. Gatilho: entrada na vértice Wealth.

The Dark Choice

Marketing. Claro/roxo → escuro Finance Noir. A página mergulha no tom rigoroso de negócios para o fechamento. Gatilho: fim da Fase 3 da landing.

08 // REGRAS E BOAS PRÁTICAS

O que NÃO deve ser feito

  • Não use ciano como cor de vértice: o ciano é exclusivo da holding VERTIC.
  • Não troque a haste do V-Node de cor: a haste é sempre Slate Gray #71717a; só o nó muda de cor.
  • Não confunda os dois marks: Mark A (V único) para logotipos; Mark B (triplo) só para preloader, loading e watermark.
  • Não use o dourado claro em fundo claro (nem o ciano escuro em fundo escuro): respeite a regra de contraste metal-fundo.
  • Não distorça nem achate o logotipo; mantenha o aspect-ratio travado.

// Estudos de cor arquivados: as 5 propostas exploratórias (Neon Cyan, Luxury Gold, Royal Blue, Crimson, Emerald) foram superadas pela arquitetura B e não devem ser usadas como marca.
